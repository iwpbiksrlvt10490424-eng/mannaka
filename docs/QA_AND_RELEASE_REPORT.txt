3. データ永続性について
==============================================================

アプリにはログイン/ログアウト機能がなく、Firebase匿名認証を使用。

[データが引き継がれる場合]
  アプリを閉じて再起動      → OK（同じUID継続）
  iPhoneを再起動して起動    → OK（同じUID継続）

[データが引き継がれない場合]
  アプリを削除して再インストール → NG（新しい匿名UIDが発行される）
  別のiPhoneで使用              → NG（端末が違うと別UID）

[保存先]
  ニックネーム・ホーム駅・年代・オンボーディング完了 → SharedPreferences（端末内）
  検索履歴・訪問記録                                 → Firebase Firestore（クラウド）

[認証フロー（auth_provider.dart）]
  1. アプリ起動 → Firebase.currentUser を確認
  2. ログイン済み → 既存UIDを使用（データ継続）
  3. 未ログイン  → signInAnonymously() で新規匿名UID取得
  4. Firebase無効 → SharedPreferencesにdevice_uidを保存してフォールバック
